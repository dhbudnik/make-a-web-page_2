PAGE & STRUCTURE

HTML defines the content and structure of the page.

HTML elements are the building rectangular blocks of a page. They describe the content so the browser knows how to show them.

Each rectangular block or box is an element created by a opening tag and a closing tag. We can find (or not) content between the tags. Tags can have attributes with values.

The Document Objective Model (DOM) defines the structure of a document and how can it be accessed and manipulated.

HTML building blocks create the structure of a page. As we will see, pages follow a tree structure where we can find brunches in the form of elements inside other elements.

Some HTML elements are:
  • head: contains metadata like title or style
  • body: it contains the content of the page
  • headings: they go from h1 to h6
  • paragraphs: defines paragraphs
  • a elements: they link an attribute value to a link text
  • images: they have the src attribute that has the address of the image
  • ordered list: ol
  • unordered list: ul
  • list elements: li
  • All elements are rectangulars
  • All the text of a page is deep in the code

TREES

Pages follow a brunching structure. This means that we can find elements inside other elements and so on. tags creates the elements in the tree

CSS: CASCADING STYLE SHEETS

CSS is the language that allow us to give style to the elements of the HTML.
  • In the "head", a style sheet must be linked so the document can access it.
  • A stong concept in CSS is not to repeat the code. For this, we use selectors.


DRY: DON'T REPEAT YOURSELF

  • Avoiding unnecesary repetition makes cleaner codes
  • Repetition is a waste: code must be mantained, so repetition is a potential source for bugs
  • Repetition calls automatization
  • Repetition makes the code harder to understand


SELECTORS

Selectors can be Tags or Classes.

  • Tag Selector Example: p{}
  • Class Selector Example: .class{attributes}

Some examples of selectors that we use in this code to avoid repetition are:

  • class selector level2
  • class selector image
  • tag selector um
